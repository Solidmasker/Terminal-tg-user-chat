# (Terminal Userbot) v1.0

Простой, легковесный и быстрый консольный мессенджер для Telegram, работающий непосредственно от имени вашего личного аккаунта (Userbot). Написан на Python с использованием библиотеки Pyrogram.

Проект позволяет переписываться в Telegram прямо из любимого эмулятора терминала на Linux, отправлять текстовые сообщения, картинки и быстро переключаться между активными чатами.

## Особенности

* **Полная интеграция с терминалом**: Читайте и пишите сообщения без графического интерфейса.
* **Безопасность**: Ваши личные ключи API (`API_ID` и `API_HASH`) хранятся локально в файле конфигурации и никогда не попадут в публичный доступ.
* **Удобное управление чатами**:
  * `/chat <ID или @username>` — переключить активный диалог.
  * `/me` — быстрое переключение на Избранное (Saved Messages).
  * `/who` — узнать, с кем вы сейчас общаетесь.
  * `/mchats` — вывод списка предустановленных быстрых контактов.
  * Быстрый переход по кнопкам `1`, `2`, `3`, `4` к закрепленным чатам.
* **Отправка медиа**: Просто введите путь к файлу изображения (`.png`, `.jpg`, `.gif`), чтобы отправить его собеседнику.

##  Установка и запуск (на примере Arch Linux)

### 1. Подготовка окружения

Рекомендуется устанавливать зависимости в изолированное виртуальное окружение, чтобы не засорять системные пакеты:

```bash
# Создаем виртуальное окружение
python -m venv ~/.local/share/tg-userbot-venv

# Устанавливаем необходимые библиотеки
~/.local/share/tg-userbot-venv/bin/pip install pyrogram tgcrypto

2. Настройка конфигурации

Создайте файл конфигурации по пути ~/.config/tg-bot/config.conf:
Bash

mkdir -p ~/.config/tg-bot
nano ~/.config/tg-bot/config.conf

Вставьте в него ваши данные. Получить API_ID и API_HASH можно на официальном сайте my.telegram.org в разделе API development tools:
Ini, TOML

API_ID="ВАШ_API_ID"
API_HASH="ВАШ_API_HASH"
FRIEND_CHAT_ID="ID_ИЛИ_USERNAME_ДРУГА_ПО_УМОЛЧАНИЮ"

3. Установка скрипта

Поместите исполняемый файл скрипта в вашу локальную папку бинарников:
Bash

cp tg-userchat ~/.local/bin/tg-userchat
chmod +x ~/.local/bin/tg-userchat

(Убедитесь, что путь ~/.local/bin добавлен в вашу переменную $PATH).
🏁 Первый запуск

Запустите утилиту в терминале:
Bash

tg-userchat

При первом запуске Pyrogram попросит ввести ваш номер телефона и код подтверждения, который придет в официальный клиент Telegram. После успешной авторизации сессия сохранится локально в ~/.config/tg-bot/my_account.session, и последующие запуски будут происходить мгновенно.
     Безопасность

Файлы .session, .session-journal, а также локальные файлы конфигурации config.conf добавлены в .gitignore и никогда не должны отправляться в публичный репозиторий GitHub.
